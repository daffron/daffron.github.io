    • Javascript Scope

    In short, Scope in Javascript refers to to visibilty of variables.
    Variables defined inside functions are said to have a local scope, meaning they can only be accessed inside the function.
    Variables defined on there own have a global scope and can be accessed from anywhere in that JS file.
    A good practice, instead of having a lot of global variables, is to have values declared inside objects. E.G

    Instead of:
    var apples = 6;
    var bananas = 9;
    var oranges = 2;

    You can put them in a global object:

    var fruit = {
    apples: 6,
    bananas: 9,
    oranges: 2
    }

    And these can be accessed globally via dot or bracket notation:

    var x = fruit.apples;
