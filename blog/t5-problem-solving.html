    A tale of problem solving
    One problem that got me thinking in the last couple weeks is the FizzBuzz challenge from last week, In which I used the following problem solving techniques:

    • Trying Something
    • Reading error messages
    • Googling
    The process was frustrating but in the end rewarding, I learnt from this a deeper knowledge of JS syntax.
    Explain to your non-tech friend:
    • .map() - Applies a function to every element in an array.
    • .filter() - Returns a new array after a function has been applied to it, which the values that returned true.
    • .reduce() - Reduces an array to a single value.
