    • Javascript Basics

    The largest differences between Javascript compared to the likes of HTML and CSS is that it can be used to program, store data and provide interactivity.

    Two elements of JS that can be described as a everyday life process are 'control flow and loops': Lets say you have finished your dinner are about to load the dishwasher, once you open if you have two options.
    IF it is full - then you would empty it,
    Otherwise or ELSE you would just put your plate in.
    That was similar to a control flow statement.
    Now a basic loop may be similar to a teacher doing a role call, they start at position 0, and for each entry they will call out the name and wait for a reply. The teach will continue the process until all the names have been called out.
    Accessing data from Arrays differs to Objects due to arrays being accessed by declaring the position in the array.
    Objects are accessed by Dot or Bracket notation, using key and object names.
    Functions, in short, are blocks of code which run when 'called', they are useful as they make it so only required code is read by the browser.
