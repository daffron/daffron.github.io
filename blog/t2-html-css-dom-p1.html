    HTML

    HTML (Hyper-Text-Markup-Language) is basically the structure or the backbone of the page. This defines the elements and content. Think of it like a frame of a house.

    CSS

    CSS Cascading Style Sheets are used to define the style of the content. This would be the process of picking which cladding and wallpaper you want on the house.

    DOM

    The Document Object Model is what your browser uses to assemble the page, so like how you would now give the plans of your house to a builder.

    What is boxifying a design?

    This is simply the process of taking a website design and breaking it down into the boxes it comprises off, making it much easier to build.

    The Box Model

    The box model is the name of how all HTML, CSS documents work. Everything element in these documents is considered a box
